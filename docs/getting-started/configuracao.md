# Configuração

Passos rápidos de configuração após a instalação:

- Verifique o `PATH` e o acesso às CLIs desejadas (claude, codex, copilot).
- Garanta permissão de leitura/escrita na pasta `skills/` do repositório.
- Opcional: crie aliases ou funções no seu shell para facilitar chamadas às CLIs.

 Se for usar ambientes isolados (Docker, WSL), siga as instruções específicas em `./instalacao.md`.
