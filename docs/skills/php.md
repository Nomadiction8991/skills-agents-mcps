# php

Objetivo: atuar como especialista PHP sênior para implementar código em PHP 8.3+ (Laravel, Symfony, Yii), com tipagem estrita, PSR, segurança e validação via PHPStan e testes.

O que faz:
- Orienta arquitetura, modelagem de domínio com tipos fortes, padrões de entrega (DTOs, serviços, endpoints) e exigência de análise estática e testes antes de entrega.

Regras principais:
- `declare(strict_types=1)` em todos os arquivos.
- Rodar `PHPStan` (nível 9) e testes (`PHPUnit`/`Pest`) antes de finalizar.
- Evitar `mixed`, proteger contra injeção, e não deixar debug em produção.
