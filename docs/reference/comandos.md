# Comandos e fluxos

Este documento descreve comandos e fluxos comuns para trabalhar com este repositório de skills.

Exemplos rápidos:

- Clonar o repositório: `git clone git@github.com:Nomadiction8991/skills-agents-mcps.git`
- Listar a pasta `skills`: `ls -la skills/`
- Verificar links criados pelo instalador: `ls -ld ~/.codex/skills ~/.claude/skills ~/.copilot/skills`

Para fluxos detalhados, veja `../getting-started/instalacao.md` e os `SKILL.md` dentro de `skills/`.
