# Exemplos Compactos

Estes são exemplos de decisões, não templates para copiar.

## Exemplo 1: Barra lateral + área principal
Prefira:
- o mesmo fundo base para a barra lateral e a área principal
- divisor sutil para separar
- contraste mais forte reservado para a navegação ativa, não para a barra inteira

Por quê:
- mantém o produto como um único espaço, não dois mundos desconectados

## Exemplo 2: Dropdown sobre cartão
Prefira:
- dropdown um nível de elevação acima do cartão pai
- borda ligeiramente mais forte que a superfície pai, se necessário

Por quê:
- preserva camadas e contenção

## Exemplo 3: Estilo de input
Prefira:
- fundo do controle levemente rebaixado
- estado de foco claro, mas contido
- tokens dedicados para controles

Por quê:
- inputs devem parecer interativos sem dominar as superfícies do layout

## Exemplo 4: Apresentação de métricas
Evite reutilizar o mesmo cartão de KPI por hábito.

Em vez disso, escolha a estrutura pelo significado:
- métrica hero quando um número domina
- métrica inline quando apoia outra tarefa
- bloco de tendência quando a mudança importa mais que o valor absoluto
- badge compacto quando o status importa mais que a escala
